# 2D Discrete Traffic Model

## Essential Features Checklist
* 30x30 grid, no wrapping. ✓
* cars with their own goals. ✓
* Simple timestep system which moves cars at set times e.g. 1 movement per second.
* Collision system to stop cars from moving into the same space.
* Pathfinding algorithm implementation, start with A*, make sure it's a simplified version which disallows diagonal movement. ✓
* Either real-time sim or a collation of snapshots which are compiled into a GIF.
